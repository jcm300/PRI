Resumo

O objetivo neste trabalho é usar o Node.js para criar um servidor no qual o website permite a submissão de ficheiros.

Após a submissão, o ficheiro deve aparecer na lista dos ficheiros submetidos.

A parte da submissão como a listagem dos ficheiros deve aparecer na mesma página Web (SPA - Single-page application).

Deve ser possível observar os ficheiros ao carregar no nome dos mesmos.

Na criação do servidor deve ser usado o Express Generator.
