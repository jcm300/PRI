Resumo

O objetivo neste trabalho é usar o Node.js para criar um servidor no qual o website permite o registo de teses.

A tese deve possuir o título, a data e o seu tipo: Mestrado ou Doutoramento.

Já em relação ao aluno deve possuir o seu nome, o número de identificação, o curso e a instituição.

O website deve possuir uma página principal com todos os serviços disponíveis, sendo um dos serviços, listar todas as teses já registadas e outro dos serviços é obviamente, o registo de teses.
