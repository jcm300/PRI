Resumo

O objetivo neste trabalho é usar o Node.js para criar um servidor de modo a suportar um Website a partir dos varios ficheiros JSON gerados do arquivo-musica-digital.xml.

De modo a criar o Website é criado um índice a partir dos ficheiros JSON.

Para além disso, uma das tarefas é usar templates para as páginas, de modo a que para apresentar uma página Web, a mesma é obtida através da inserção dos dados presentes nos JSON's nas templates. Para tal é usado o Pug.

Por fim, o servidor deve receber endereços e responder ao cliente de forma correta, consoante o endereço fornecido.
