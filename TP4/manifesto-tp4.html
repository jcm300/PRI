Resumo

O objetivo neste trabalho é usando o Node.js, criar um servidor de modo a suportar um Website (neste caso o Website criado a partir de arquivo-musica-digital.xml usando uma stylesheet).

O servidor deve receber respostas e consoante o URL fornecido deve apresentar a página correta, bem como claro, o endereçamento do Website deve funcionar corretamente.
